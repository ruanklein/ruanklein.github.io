# ruan.sh

Personal portfolio website built with Svelte and Tailwind CSS.

## Features

- **Minimalist Design**: Pure black (#000) and white (#fff) color scheme
- **Internationalization**: English and Brazilian Portuguese support
- **Project Showcase**: Interactive modals with detailed project stories
- **Responsive**: Mobile-first design with smooth transitions
- **Dark Mode Only**: True dark theme without compromise

## Tech Stack

- [Svelte 5](https://svelte.dev/) - Reactive framework
- [Tailwind CSS 4](https://tailwindcss.com/) - Utility-first CSS
- [Vite](https://vitejs.dev/) - Build tool
- [i18n-js](https://github.com/fnando/i18n) - Internationalization
- [lucide-svelte](https://lucide.dev/) - Icon library

## Development

```bash
# Install dependencies
yarn install

# Start dev server
yarn dev

# Build for production
yarn build

# Preview production build
yarn preview
```
